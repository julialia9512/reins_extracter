# Simple Upload to GitHub

Since Git isn't working in your PowerShell, here's the easiest way to upload your project to GitHub:

## Method 1: Upload via GitHub Website (Easiest)

1. **Go to GitHub and create the repository:**
   - Visit: https://github.com/new
   - Repository name: `reins_extracter`
   - Choose Public or Private
   - **DON'T** check any boxes (no README, no .gitignore, no license)
   - Click "Create repository"

2. **Upload files via GitHub web interface:**
   - On your repository page, click "uploading an existing file"
   - Select all your project files:
     - app.py
     - Readme.md
     - requirements.txt
     - .gitignore
     - GITHUB_SETUP.md
     - INSTALL_GIT.md
   - Drag and drop or click to select all files
   - Enter commit message: "Initial commit"
   - Click "Commit changes"
